#  Weather Card App

A lightweight weather application built with **HTML, CSS, and JavaScript.**  
It fetches real‑time weather data from the **OpenWeatherMap API** and displays it in a neat card layout.

---

# Features
- Search weather by city name
- Shows temperature, feels like, humidity, wind, sunrise & sunset
- Weather icons for better visualization
- Responsive card design

---

# Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- OpenWeatherMap API  

---

# Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/weather-card-app.git
2. Get your API key from OpenWeatherMap.
3. Open weather.js and replace the placeholder with your key:
    **const API_KEY = "YOUR_API_KEY_HERE";**
4. Run the project using a local server (e.g., VS Code Live Server).
5. Open index.html in your browser and search for any city.
